## ✨ Features

- **Side-by-Side Playground:** Split-pane interface featuring theory on the left and a live execution editor (Ace Editor) on the right.
- **In-Browser Execution Engine:** Safely compiles and executes user-written JavaScript (via eval traps) and HTML/CSS (via dynamic iframes) inside the browser.
- **Client-Side Authentication:** Simulates a real database using the browser's native `localStorage` to handle user registration, encrypted login state, and secure session management.
- **Progress Tracking Analytics:** Dashboard UI that calculates and saves the user's course advancement logically across sessions.
- **Premium Design System:** Linear-inspired custom aesthetics featuring dark mode, glassmorphism UI cards, glowing gradients, and CSS animated meshes without relying on Tailwind or Bootstrap.

## 📚 Curriculum & Concepts Taught

This platform comes pre-loaded with comprehensive mock courses that validate user input dynamically:

### 🟡 JavaScript Fundamentals
1. **Introduction to Variables** (`let`, `const`, and block scope)
2. **Primitive Data Types** (`Strings`, `Numbers`, `Booleans`)
3. **Basic Operations** (Mathematical execution and logic)
4. **Conditional Logic** (`if / else` flows)
5. **Arrays** (Creating, storing, and accessing list elements)
6. **For Loops** (DRY logic execution)
7. **Functions** (Reusable blocks, parameters, and returns)

### 🟠 HTML & CSS UI Design
1. **Hello HTML** (Understanding elements, tags, and structure)
2. **Attributes & Media** (`href` links and Image `src`)
3. **Structured Lists** (`<ul>`, `<ol>`, and `<li>`)
4. **CSS Basics** (`<style>` blocks, colors, backgrounds)
5. **DOM Targeting** (Utilizing CSS `Classes` and `IDs`)

## 🚀 How to Run Locally

You don't need Node.js, npm, or any modern build tools to run this application!

1. Clone the repository:
   \`\`\`bash
   git clone https://github.com/PrimeSiddhesh/CodeMastery.git
   \`\`\`
2. Navigate to the project folder:
   \`\`\`bash
   cd CodeMastery
   \`\`\`
3. Open `index.html` in your web browser, or use a local development server like VS Code's **Live Server** extension for real-time updates.
